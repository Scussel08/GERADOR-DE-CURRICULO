Gerador de Currículos em PHP

Este projeto foi desenvolvido como parte da Atividade Prática Orientada (APO) da disciplina Fundamentos de Programação para Internet do curso de Engenharia de Software da UNIPAR - Universidade Paranaense.

O sistema tem como objetivo gerar um currículo personalizado a partir dos dados inseridos pelo usuário, utilizando tecnologias web modernas e boas práticas de desenvolvimento.

Objetivo do Projeto

O Gerador de Currículos Online permite ao usuário preencher um formulário com informações pessoais, idade (calculada automaticamente) e experiências profissionais, gerando um currículo dinâmico pronto para download ou impressão.

Tecnologias Utilizadas

HTML5 — estrutura do conteúdo

CSS3 — estilização e responsividade

JavaScript (jQuery) — interatividade e adição dinâmica de campos

PHP 8 — processamento e geração do currículo

Bootstrap — layout moderno e responsivo

Git/GitHub — versionamento e hospedagem do código

Figma — prototipagem das telas

Como Executar o Projeto

Instale o XAMPP (ou outro servidor PHP local).

Copie a pasta do projeto para o diretório:

C:\xampp\htdocs\


Inicie o Apache no painel do XAMPP.

No navegador, acesse:

http://localhost/GERADOR-DE-CURRICULO/


Preencha o formulário e clique em "Gerar Currículo".
O sistema exibirá as informações formatadas e abrirá a opção para imprimir ou salvar em PDF.

Funcionalidades Principais

Cálculo automático da idade com JavaScript.

Adição dinâmica de novas experiências profissionais.

Geração automática do currículo com PHP.

Layout simples, responsivo e intuitivo.

Impressão e download direto pelo navegador.
